# 🌍 Bharat Climate Twin

> **AI-Powered Digital Twin for India's Climate**

An intelligent climate intelligence platform that combines satellite observations, weather data, AI-powered insights, and scenario simulations into a unified Digital Twin for India.

🌐 **Live Demo:** https://climate-twin-xeg4.vercel.app

---

## 📌 Overview

Bharat Climate Twin is a full-stack AI-powered platform designed to help Farmers, Scientists, and Policymakers understand climate conditions through interactive dashboards, AI-generated insights, and real-time climate intelligence.

The platform integrates multiple climate datasets into a single Digital Twin, enabling users to monitor weather patterns, assess climate risks, simulate future warming scenarios, and receive role-specific recommendations.

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/malhotrapearl/Climate-twin-main.git
```

Frontend

```bash
npm install

npm run dev
```

Backend

```bash
cd backend

pip install -r requirements.txt

uvicorn server:app --reload
```

---

## 🔑 Environment Variables

Create a `.env` file inside the backend directory.

```env
MONGO_URL=

DB_NAME=

JWT_SECRET=

GROQ_API_KEY=

CORS_ORIGINS=
```
